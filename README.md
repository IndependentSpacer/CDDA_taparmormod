## 📦 Features

### Hub 01 Tactical Armored Platesuit
- Full-body composite armor system  
- Constructed from UHMWPE-steel plates with kevlar backing  
- Strong protection against melee and low-caliber ballistic threats  
- Integrated plate carrier system for additional armor plates  

### Hub 01 Helmet
- Fully enclosed composite helmet  
- Integrated powered air-purifying respirator (PAPR)  
- Environmental protection with modular attachment points  

### Hub 01 Mercenary Profession
- Starts with the full armor set and standard soldier equipment  

## 📁 Installation

1. Download the mod  
2. Place the mod folder into: Your cdda folder/data/mods/
